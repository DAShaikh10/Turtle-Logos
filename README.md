# Introduction
This repository contains different logo's drawn using python `turtle` module.

## List of logo's
1. Windows 10

    > **Usage**:   
    > ```powershell
    > python windows_10_logo.py -t color
    > ``` 
    > or  
    > ```powershell
    > python windows_10_logo.py --theme color
    > ``` 
    > *color:* (**Default:** White, **Alternate:** Blue)
